󹃮 ㄎㄞˇ

[名](古文)骨間肉。音苦等切，迥韻。康典引玉篇云今作「肯」。讀「 ㄎㄞˇ」的「󹃮」，康典引字林云著骨肉。(台)按： 無 ㆠㄜˊ 論 ㄌㄨㄣ˫ 是 ㄒㄧ˫ 「 骨 ㄍㄨㆵ 間 ㄍㄢ 肉 ㆠㄚㆷ 、 著 ㄉㄧㄠˊ 骨 ㄍㄨㆵ 肉 ㆠㄚㆷ 」，「󹃮」字今已罕用，字併入【肯】，國音今用「 ㄎㄣˇ」、「 ㄎㄥˇ」，不存「 ㄎㄞˇ」音。按：私謂：當「骨上肉」解的「肯」字，字形可以還給「 󹃮 ㄎㄞˇ 」(古語)。「肯不肯」的「肯」讀「 ㄎㄥˇ」，合於反切，切腳。相當於台語的「 ㄎㄧㄥˋ」，(又音) ㄎㄣˇ相當於漳州系的「 ㄎㄧㄣˋ」。今語的「肯」，不需再存古形「󹃮」及「 ㄎㄞˇ」音。「肯」的閩南音似曾用「 ㄎㆮ」形的音。
